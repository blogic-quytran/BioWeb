# Shop Profile UI

Giao diện HTML/CSS/JS có backend Python + SQLite để lấy dữ liệu hiển thị.

## Chạy dự án
1) Cài dependencies:
	- pip install -r requirements.txt
2) Chạy server:
	- python app.py
3) Mở trình duyệt:
	- http://localhost:5500

## Ghi chú
- Ảnh đang là placeholder (khối màu). Hãy thay bằng ảnh thật khi có.
- Dữ liệu mẫu nằm trong SQLite: data.db (tự tạo khi chạy lần đầu).
- Có thể tải ảnh từ máy khi thêm sản phẩm; file sẽ lưu ở thư mục uploads.
