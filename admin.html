Deploy Vercel (Flask API)
1) Push project lên GitHub.
2) Vercel → New Project → Import repo.
3) Root Directory: d:\WEB (hoặc repo root).
4) Thêm file cần thiết: vercel.json + api/index.py để chạy Flask.
5) Deploy.
